# AI in Economics & Financial Econometrics

**Автор**: Глеб Бурин, группа Б05-312

## Финансовая эконометрика (Financial Econometrics)

* **[hw1](./Financial%20Econometrics/hw1)**: Симуляция процессов стохастической волатильности и лог-цен финансового актива с использованием модели Хестона (Heston model).
* **[hw3](./Financial%20Econometrics/hw3)**: Обработка высокочастотных финансовых временных рядов (OHLCV) на примере акций Bank of America (BAC), вычисление лог-доходностей и визуализация динамики актива.

## Искусственный интеллект в экономике (AI in Economics)

* **[hw1 (Name Generator)](./hw1)**: Построение генеративной модели на основе матрицы переходных вероятностей (n-граммной модели) для синтеза текста. Использован базовый тензорный аппарат PyTorch.
* **[hw2 (Karpathy minigrad)](./hw2)**: Имплементация системы автоматического дифференцирования и алгоритма обратного распространения ошибки (backpropagation) с нуля (Bare Python). Включает функционал построения графов вычислений.
* **[hw3 (Fashion-MNIST)](./hw3)**: Разработка полного пайплайна обучения нейронной сети для задачи классификации изображений. Реализована нормализация данных, подготовка батчей через DataLoader и оценка метрик модели.
* **[hw4 (Contextual Bandits)](./hw4)**: Решение задачи многоруких бандитов с контекстом (Contextual Bandits) для оптимизации системы рекомендаций. Реализован алгоритм LinUCB с применением гребневой регрессии для оценки математического ожидания выигрыша, проведено сравнение с неконтекстными подходами.
* **[hw5 (Adversarial Attack)](./hw5)**: Исследование уязвимости ML-моделей к состязательным атакам. Произведено вычисление оптимального вектора атаки на веса логистической регрессии для искажения результатов бинарной классификации подвыборки датасета MNIST.
